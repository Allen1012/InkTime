# PCB 设计基础知识

> 面向硬件零基础的软件开发者，以墨水屏转接板为实例讲解。
> 看完这篇文档，你能看懂任何简单的 PCB 设计图。

---

## 一、PCB 是什么

PCB = Printed Circuit Board（印刷电路板）。

你拆开任何电子设备，里面那块绿色（或蓝色、黑色、红色）的板子就是 PCB。
它的本质是一块绝缘板（通常是玻璃纤维），上面印刷了铜线来连接各个电子元件。

```
PCB 的截面（侧视图）：

    元件      元件      元件
     │         │         │
  ───┴────┬────┴────┬────┴───  ← 顶层铜（Top Copper）：铜线走线
  ████████│█████████│████████  ← 阻焊层（Solder Mask）：绿色油漆
  ────────┴─────────┴────────  ← 基板（FR-4）：玻璃纤维绝缘板
  ███████████████████████████  ← 阻焊层（Solder Mask）
  ───────────────────────────  ← 底层铜（Bottom Copper）
```

### 为什么不直接用导线连接？

对于 2-3 个元件，用导线确实可以。但当元件多了（比如 ESP32 有几十个引脚），
导线会变成一团乱麻，而且容易接触不良。PCB 把所有连接"印刷"在板子上，
可靠、整洁、可批量生产。

## 二、PCB 的层结构

### 2.1 一块 2 层 PCB 有哪些层

你截图中的嘉立创 EDA 界面右侧列出了 PCB 设计的常规事项。
一块标准的 2 层 PCB 包含以下层：

```
从上到下：

┌─ 顶层丝印 (Top Silkscreen) ─── 白色文字，标注元件名称（J1、C1、R1）
├─ 顶层阻焊 (Top Solder Mask) ── 绿色油漆，保护铜线，焊盘处开窗露铜
├─ 顶层铜   (Top Copper) ─────── 正面的铜线走线和焊盘
├─ 基板     (FR-4 Substrate) ─── 玻璃纤维绝缘板，约 1.6mm 厚
├─ 底层铜   (Bottom Copper) ──── 背面的铜线走线和焊盘
├─ 底层阻焊 (Bottom Solder Mask) ─ 背面的绿色油漆
└─ 底层丝印 (Bottom Silkscreen) ── 背面的白色文字
```

### 2.2 每一层的作用

**铜层（Copper Layer）**

这是最核心的层。铜线就是电路中的"导线"，连接各个元件。

```
铜层上你会看到：

焊盘（Pad）：     ● ← 圆形或方形的铜区域，元件的引脚焊在这里
走线（Trace）：   ─── ← 细长的铜线，连接两个焊盘
铺铜（Pour）：    ███ ← 大面积的铜区域，通常是 GND（接地）
过孔（Via）：     ◎ ← 穿透板子的小孔，连接顶层和底层的铜线
```

**阻焊层（Solder Mask）**

就是 PCB 上那层绿色（或其他颜色）的油漆。它的作用：
- 保护铜线不被氧化
- 防止焊接时焊锡流到不该去的地方
- 只在焊盘位置"开窗"（露出铜），方便焊接

**丝印层（Silkscreen）**

白色的文字和图案，印在阻焊层上面。作用：
- 标注元件位号（J1、C1、R1）
- 标注元件方向（极性标记）
- 标注板子名称、版本号
- 标注引脚功能（VCC、GND、DIN 等）

## 三、如何看懂 PCB 设计图

### 3.1 原理图 vs PCB 图

PCB 设计有两种图，它们是同一个电路的两种表达方式：

```
原理图（Schematic）              PCB 图（Layout）
┌─────────────────┐            ┌─────────────────┐
│                 │            │                 │
│  逻辑关系图      │    转换    │  物理布局图      │
│  "谁连谁"       │  ──────→  │  "放在哪、线怎么走"│
│                 │            │                 │
│  不关心物理位置   │            │  精确到 0.01mm   │
│  只关心连接关系   │            │  就是工厂生产的图纸│
│                 │            │                 │
└─────────────────┘            └─────────────────┘
```

**原理图**：用符号表示元件，用线表示连接。像电路的"思维导图"。
**PCB 图**：元件的实际形状和位置，铜线的实际走向。像电路的"施工图"。

### 3.2 原理图中的常见符号

```
电阻：     ─[///]─  或  ─[R]─     标注：R1 10kΩ
电容：     ─||─     或  ─[C]─     标注：C1 100nF
LED：      ─▷|─                   标注：LED1
按键：     ─/ ─                   标注：SW1
连接器：   ─[█]─                  标注：J1
芯片/IC：  ┌──┐
           │  │                   标注：U1 ESP32
           └──┘
GND 符号：  ⏚  或  ▽              表示接地
VCC 符号：  ─── 或 带箭头          表示电源正极
NC：       ✕                      Not Connected，不连接
```

### 3.3 PCB 图中的常见元素

以你截图中的嘉立创 EDA 界面为例：

```
你在 PCB 图中会看到的东西：

红色线/区域 ──→ 顶层铜（Top Copper）
蓝色线/区域 ──→ 底层铜（Bottom Copper）
绿色区域    ──→ 阻焊层开窗（焊盘露铜的地方）
白色文字    ──→ 丝印（元件标注）
黄色线      ──→ 板框（Board Outline）
圆形带十字  ──→ 过孔（Via），连接顶层和底层
圆形/方形   ──→ 焊盘（Pad），焊接元件的地方
```

### 3.4 看懂你截图中的 PCB

你发的那张嘉立创截图是一个"波形产生与变换设计电路"的 PCB，比我们的转接板复杂得多。
但基本元素是一样的：

```
截图中标注的内容解读：

"槽孔"        → 板子上的非圆形孔（比如 USB 接口的固定孔）
"放置文本"     → 在丝印层添加文字标注
"放置板框"     → 画 PCB 的外形轮廓
"泪滴添加"     → 走线和焊盘连接处加一个水滴形过渡，增强可靠性
"铺铜"         → 大面积填充铜（通常是 GND）

右侧 "PCB设计常规事项"：
  1. 规则设置   → 设置最小线宽、最小间距等制造参数
  2. 设计检查   → DRC（设计规则检查），检查有没有违规
  3. 网格与对齐 → 元件和走线对齐到网格，保持整齐
  4. 图层管理   → 切换显示/隐藏各个层
  5. 3D 模型    → 预览 PCB 的 3D 效果
  6. 生产文件   → 导出 Gerber 文件给工厂
```

## 四、PCB 设计中的关键概念

### 4.1 焊盘（Pad）

焊盘是 PCB 上用来焊接元件引脚的铜区域。分两种：

```
贴片焊盘（SMD Pad）：          通孔焊盘（Through-Hole Pad）：
只在一面有铜                    穿透整块板子

  ┌───┐                           ┌─●─┐
  │ ▓ │ ← 铜区域                  │ │ │ ← 孔 + 两面都有铜环
  └───┘                           └─●─┘
  只在顶层或底层                   顶层和底层都有

用于：FPC 连接器、电容、电阻      用于：排针、直插元件
```

我们的转接板上：
- FPC 连接器和电容用**贴片焊盘**（焊在正面）
- 排针用**通孔焊盘**（引脚穿过板子，在背面焊接）

### 4.2 走线（Trace）

走线就是 PCB 上的铜"导线"，连接两个焊盘。

```
走线的关键参数：

线宽（Width）：
  ─── 0.25mm ← 信号线，电流小
  ━━━ 0.40mm ← 电源线，电流大

间距（Clearance）：
  ───  ───
    ↑
  两根走线之间的最小距离，通常 ≥ 0.15mm
  太近会导致工厂做不出来或者短路
```

### 4.3 过孔（Via）

过孔是一个穿透 PCB 的小孔，内壁镀铜，用来连接顶层和底层的走线。

```
俯视图：        截面图：

  ◎              顶层铜 ──┐
                          │ ← 镀铜孔壁
                 底层铜 ──┘

用途：当顶层走线被其他走线挡住走不通时，
     通过过孔"钻"到底层绕过去，再"钻"回来。
```

我们的转接板很简单，可能不需要过孔。但如果走线交叉了，就需要用过孔绕到背面。

### 4.4 铺铜（Copper Pour）

在 PCB 的空白区域填充大面积的铜，通常连接到 GND（接地）。

```
没有铺铜：                    有铺铜：
┌──────────────┐            ┌██████████████┐
│  ───  ───    │            │██───██───████│
│              │            │██████████████│
│  ───         │            │██───█████████│
│              │            │██████████████│
└──────────────┘            └██████████████┘
                             █ = GND 铜皮
```

好处：
- 减少电磁干扰（EMI）
- 提供良好的 GND 回流路径
- 散热
- 看起来更专业

### 4.5 丝印（Silkscreen）

白色的文字和图案，帮助你焊接时识别元件位置和方向。

```
好的丝印设计：

  ┌────────────────────────────┐
  │  J1                        │  ← 元件位号
  │  ┌──────────────────────┐  │
  │  │  FPC 50P              │  │  ← 元件描述
  │  └──────────────────────┘  │
  │                            │
  │  [C1]                      │  ← 电容位置
  │                            │
  │  VCC GND DIN CLK CS DC RST BUSY │  ← 引脚功能标注
  │  ┌──┬──┬──┬──┬──┬──┬──┬──┐│
  │  │1 │2 │3 │4 │5 │6 │7 │8 ││  ← pin 编号
  │  └──┴──┴──┴──┴──┴──┴──┴──┘│
  │                            │
  │  InkTime EPD Adapter v1.0  │  ← 板子名称和版本
  └────────────────────────────┘
```

在丝印上标注引脚功能非常重要——焊接和接线时不用反复查原理图。

## 五、立创 EDA 操作指南

### 5.1 界面布局

```
┌─────────────────────────────────────────────────┐
│  菜单栏：文件 编辑 放置 布线 设计 制造 ...        │
├──────┬──────────────────────────────┬────────────┤
│      │                              │            │
│ 元件 │                              │  属性面板  │
│ 库   │      主编辑区域               │            │
│      │    （原理图或 PCB）           │  显示当前  │
│      │                              │  选中元件  │
│      │                              │  的参数    │
│      │                              │            │
├──────┴──────────────────────────────┴────────────┤
│  状态栏：坐标、网格大小、当前层                     │
└─────────────────────────────────────────────────┘
```

### 5.2 原理图编辑器常用操作

| 操作 | 快捷键/方法 | 说明 |
|------|-----------|------|
| 放置元件 | 从左侧元件库拖入 | 搜索型号或关键词 |
| 连线 | W 键，或点击工具栏的画线 | 点击起点 → 点击终点 |
| 移动元件 | 选中后拖动 | 连线会跟着移动 |
| 旋转元件 | 选中后按 R | 每次旋转 90° |
| 添加电源符号 | 搜索 "VCC" 或 "GND" | 从元件库拖入 |
| 添加 NC 标记 | 搜索 "NC" | 标记不连接的引脚 |
| ERC 检查 | 菜单 "设计" → "ERC" | 检查电气规则错误 |

### 5.3 PCB 编辑器常用操作

| 操作 | 快捷键/方法 | 说明 |
|------|-----------|------|
| 画板框 | 切换到板框层，画矩形 | 定义 PCB 外形 |
| 摆放元件 | 拖动元件到目标位置 | 先摆好再布线 |
| 布线 | W 键，或点击工具栏的布线 | 点击起始焊盘 → 沿路径点击 → 点击终点焊盘 |
| 切换层 | 布线时按 V | 自动添加过孔，切换到另一层继续走 |
| 设置线宽 | 布线前在属性面板设置 | 电源线 0.4mm，信号线 0.25mm |
| 铺铜 | 菜单 "放置" → "铺铜" | 画一个区域，设置网络为 GND |
| DRC 检查 | 菜单 "设计" → "DRC" | 检查设计规则错误 |
| 3D 预览 | 菜单 "视图" → "3D" | 看板子的立体效果 |

### 5.4 设计规则设置

在开始布线之前，先设置设计规则（菜单 "设计" → "设计规则"）：

| 规则 | 推荐值 | 说明 |
|------|--------|------|
| 最小线宽 | 0.15mm | 嘉立创能做到的最小值 |
| 最小间距 | 0.15mm | 走线之间、走线和焊盘之间 |
| 最小过孔外径 | 0.45mm | 过孔的铜环直径 |
| 最小过孔内径 | 0.25mm | 过孔的钻孔直径 |
| 最小阻焊开窗 | 0.05mm | 焊盘比阻焊开窗大多少 |

这些值是嘉立创的标准工艺能力，设置好后 DRC 检查会按这些规则报错。

## 六、DRC 检查详解

DRC = Design Rule Check（设计规则检查）。相当于代码的 lint 工具，帮你找出设计中的问题。

### 6.1 常见 DRC 错误

| 错误类型 | 含义 | 如何修复 |
|---------|------|---------|
| Clearance violation | 两个铜区域太近 | 移开走线或元件，增大间距 |
| Unconnected net | 有焊盘没连上 | 补画走线连接 |
| Short circuit | 两个不同网络的铜短路了 | 删除多余的走线或铺铜 |
| Minimum width violation | 走线太细 | 加粗走线 |
| Drill size too small | 钻孔太小 | 加大过孔尺寸 |

### 6.2 DRC 通过 = 可以打板

当 DRC 检查 0 错误时，说明你的设计满足工厂的制造能力，可以放心导出 Gerber 打板了。

## 七、封装（Footprint）详解

### 7.1 什么是封装

封装 = 元件在 PCB 上的"脚印"。它定义了焊盘的形状、大小、间距。

同一个电容（比如 100nF），可以有不同的封装：

```
0402 封装：  ┌─┐   1.0mm x 0.5mm  ← 非常小，手焊困难
             └─┘

0603 封装：  ┌──┐   1.6mm x 0.8mm  ← 手焊可行（我们用这个）
             └──┘

0805 封装：  ┌───┐  2.0mm x 1.25mm ← 手焊容易
             └───┘

1206 封装：  ┌────┐ 3.2mm x 1.6mm  ← 手焊最容易，但占空间大
             └────┘
```

### 7.2 我们用到的封装

| 元件 | 封装名称 | 焊接方式 | 难度 |
|------|---------|---------|------|
| FPC 连接器 | 厂商特定封装 | 贴片（SMD） | 较难（0.5mm 间距） |
| 100nF 电容 | 0603 | 贴片（SMD） | 简单 |
| 8pin 排针 | 2.54mm 直插 | 通孔（THT） | 最简单 |

## 八、从设计到实物的完整流程总结

```
第 1 步：画原理图
  在立创 EDA 中放置元件、连线
  运行 ERC 检查，0 错误
        ↓
第 2 步：转 PCB
  原理图 → PCB，元件自动导入
        ↓
第 3 步：摆放元件
  把元件拖到合适的位置
        ↓
第 4 步：画板框
  定义 PCB 的外形和大小
        ↓
第 5 步：布线
  用铜线连接各个焊盘
  电源线粗（0.4mm），信号线细（0.25mm）
        ↓
第 6 步：铺铜
  空白区域铺 GND 铜皮
        ↓
第 7 步：添加丝印
  标注元件名称、引脚功能、板子名称
        ↓
第 8 步：DRC 检查
  0 错误 → 可以打板
        ↓
第 9 步：导出 Gerber
  "制造" → "生成制造文件"
        ↓
第 10 步：上传到嘉立创
  选参数、下单、付款
        ↓
第 11 步（可选）：SMT 贴片
  额外上传 BOM + 坐标文件
  工厂帮你焊贴片元件
        ↓
第 12 步：收货
  3-5 天后收到 PCB（或已贴片的半成品）
        ↓
第 13 步：手焊剩余元件
  焊排针等直插元件
        ↓
第 14 步：测试
  万用表检查 → 上电 → 烧录固件 → 验证功能
```
