# 微雪 7.3 寸 6 色墨水屏 (E) 转接板设计方案

> 目标：设计一块最简转接板，把微雪 7.3 寸 6 色裸屏的 50pin FPC 排线转成 8pin 排针，
> 用杜邦线连接 ESP32-S3 开发板驱动墨水屏。
>
> 适合硬件零基础的开发者，每一步都有详细解释。

---

## 一、这块板要做什么

### 1.1 问题

微雪 7.3 寸 6 色裸屏背面伸出一根 FPC 软排线（橙色、扁平、50 个触点、0.5mm 间距）。
这根排线没法直接连到 ESP32 开发板上，因为：

- ESP32 开发板上是 2.54mm 间距的排针孔
- FPC 排线是 0.5mm 间距，比排针密 5 倍
- FPC 排线需要专用的卡扣式连接器才能固定

### 1.2 解决方案

做一块小 PCB（约 30mm x 20mm），上面放：

1. 一个 50pin FPC 连接器 —— 接墨水屏的排线
2. 一个 100nF 去耦电容 —— 稳定电源
3. 一排 8pin 2.54mm 排针 —— 用杜邦线连 ESP32

板子上的铜线走线把 FPC 连接器中有用的 8 个 pin 连到排针上，其余 42 个 pin 不接。


### 1.3 整体连接关系

```
墨水屏 (7.3寸 6色)
  │
  │  50pin FPC 软排线（橙色，从屏幕背面伸出，很短约2-3cm）
  │
  ↓
┌──────────────────────────────────┐
│         你的转接板                 │  ← 这就是我们要做的东西
│                                  │
│  ┌────────────────────────────┐  │
│  │  50pin FPC 连接器 (J1)     │  │  ← 卡扣式，掀开翻盖插入排线
│  └────────────────────────────┘  │
│           │                      │
│     PCB 铜线走线（只取 8 根）     │
│           │                      │
│       [100nF 电容 (C1)]          │  ← VCC 和 GND 之间
│           │                      │
│  ┌──┬──┬──┬──┬──┬──┬──┬──┐     │
│  │V │G │DI│CK│CS│DC│RS│BU│     │  ← 8pin 排针 (J2)
│  └──┴──┴──┴──┴──┴──┴──┴──┘     │
│                                  │
└──────────────────────────────────┘
  │
  │  8 根杜邦线（母对母，约 10-20cm）
  │
  ↓
ESP32-S3 开发板
```

## 三、核心知识：FPC 连接器详解

### 3.1 什么是 FPC 连接器

FPC 连接器是一种专门用来固定 FPC 软排线的卡扣式座子。它焊在 PCB 上，有一个可以翻开的小盖子。

```
翻盖打开状态（插入排线时）：
        ┌─── 翻盖（向上掀起）
        │
┌───────┴───────────────────┐
│  ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓  │  ← 50 个金属触点（0.5mm 间距）
│                           │
│  ┌─────────────────────┐  │
│  │  PCB 焊盘            │  │  ← 焊在 PCB 上
│  └─────────────────────┘  │
└───────────────────────────┘

翻盖关闭状态（锁紧排线后）：
┌───────────────────────────┐
│  ═══════════════════════  │  ← 翻盖压下，排线被夹紧
│  ▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓  │
│  ┌─────────────────────┐  │
│  │  PCB 焊盘            │  │
│  └─────────────────────┘  │
└───────────────────────────┘
```

### 3.2 选型要点

买 FPC 连接器时，必须确认以下 4 个参数全部匹配：

| 参数 | 你需要的值 | 说明 |
|------|-----------|------|
| pin 数 | **50pin** | 必须和排线的触点数一致 |
| 间距 | **0.5mm** | 必须和排线的触点间距一致 |
| 接触方式 | **下接** | 排线触点朝下（朝 PCB 方向）插入 |
| 翻盖方向 | **后翻** 或 **上翻** | 翻盖向后或向上掀起，都可以 |

在立创商城搜索：`FPC连接器 50P 0.5mm 下接`

**注意**：如果买错了 pin 数或间距，排线插不进去。如果买错了上接/下接，排线触点方向会反，信号全部错位。

### 3.3 上接 vs 下接

```
下接（你需要的）：
  排线触点朝下 ↓ 面向 PCB
  ┌─────────────┐
  │  排线        │
  │  触点在下面▓▓│ ← 触点朝 PCB
  └──────┬──────┘
         ↓
  ┌──────────────┐
  │  FPC 连接器   │
  │  PCB 焊盘     │
  └──────────────┘

上接（不要买这个）：
  排线触点朝上 ↑ 背向 PCB
  ┌─────────────┐
  │  ▓▓触点在上面│ ← 触点背向 PCB
  │  排线        │
  └──────┬──────┘
         ↓
  ┌──────────────┐
  │  FPC 连接器   │
  └──────────────┘
```

如何判断你的屏幕排线是上接还是下接？看排线末端：触点（金色条纹）露在哪一面。
微雪 7.3 寸系列通常是**下接**（触点在排线的下面/背面），但请拿到实物后确认。

### 4.2 每个元件的作用

**J1 - FPC 连接器（50pin 0.5mm 下接翻盖式）**

- 作用：物理连接墨水屏的 FPC 排线
- 50 个 pin 中只有 8 个有用，其余不接（NC = No Connection）
- 焊在 PCB 的一端

**C1 - 去耦电容（100nF / 0.1uF，0603 封装）**

- 作用：稳定 VCC 电源，过滤高频噪声
- 连接在 VCC 和 GND 之间
- 放置位置：尽量靠近 FPC 连接器的 VCC pin
- 为什么需要：ESP32 通过杜邦线供电，线比较长会引入噪声，电容能吸收这些噪声

**J2 - 排针（2.54mm 1x8pin 直插）**

- 作用：提供标准间距的接口，用杜邦线连 ESP32
- 8 个 pin 的定义（从左到右）：

| 排针 pin | 信号名 | 方向 | 说明 |
|---------|--------|------|------|
| 1 | VCC | 输入 | 3.3V 电源，从 ESP32 的 3.3V 引脚供电 |
| 2 | GND | — | 电源地，接 ESP32 的 GND |
| 3 | DIN | 输入 | SPI 数据，ESP32 → 屏幕 |
| 4 | SCLK | 输入 | SPI 时钟，ESP32 → 屏幕 |
| 5 | CS | 输入 | SPI 片选，低电平有效 |
| 6 | DC | 输入 | 数据/命令选择，高=数据 低=命令 |
| 7 | RST | 输入 | 复位，低电平复位屏幕 |
| 8 | BUSY | 输出 | 屏幕忙状态，屏幕 → ESP32 |

## 五、最关键的一步：确认 FPC pin 映射

### 5.1 为什么这一步最重要

50pin FPC 排线中，哪个 pin 是 VCC、哪个是 DIN、哪个是 BUSY……这个映射关系搞错了，
屏幕不会亮，严重的话还可能烧屏。

### 5.2 如何获取 pin 映射

**方法 1：下载微雪驱动板原理图（推荐）**

1. 打开微雪 Wiki：https://www.waveshare.com/wiki/7.3inch_e-Paper_HAT_(E)_Manual
2. 滚动到页面底部 "Resources" 区域
3. 找到 "Driver Board Schematic" 链接，下载 PDF
4. 打开 PDF，找到 FPC 连接器部分
5. 记录每个 pin 对应的信号名

**方法 2：用万用表测量 HAT 版驱动板（如果你有的话）**

如果你手上有微雪的 HAT 版，可以用万用表的通断档，
逐个测量 FPC 连接器的 pin 和 8pin SPI 排针之间的对应关系。

### 5.3 pin 映射表模板

下载原理图后，填写这个表格：

| FPC pin 号 | 信号名 | 连接到排针 | 备注 |
|-----------|--------|-----------|------|
| pin ? | VCC | J2.pin1 | 3.3V 电源 |
| pin ? | GND | J2.pin2 | 可能有多个 GND pin，全部连起来 |
| pin ? | DIN (SDA/MOSI) | J2.pin3 | SPI 数据 |
| pin ? | SCLK (SCL/SCK) | J2.pin4 | SPI 时钟 |
| pin ? | CS (CSB) | J2.pin5 | 片选 |
| pin ? | DC (D/C) | J2.pin6 | 数据/命令 |
| pin ? | RST | J2.pin7 | 复位 |
| pin ? | BUSY | J2.pin8 | 忙状态 |
| 其余所有 pin | NC | 不接 | 不连接任何东西 |

**重要提示**：
- GND 可能有多个 pin（比如 pin 1、pin 2、pin 49、pin 50 都是 GND），全部连到 J2 的 GND
- VCC 也可能有多个 pin，全部连到 J2 的 VCC
- 有些 pin 标注为 NC 或 IOVCC 或 VDDIO，需要根据原理图判断是否需要连接

## 六、PCB 设计步骤（立创 EDA）

### 6.1 工具准备

1. 打开浏览器，访问 https://lceda.cn （立创 EDA 在线版）
2. 注册账号（免费）
3. 新建工程，命名为 "InkTime-EPD-Adapter"

### 6.2 画原理图

**第 1 步：放置 FPC 连接器**

1. 在立创 EDA 的元件库中搜索你买的 FPC 连接器型号
2. 或者搜索 "FPC 50P 0.5mm"，找到匹配的封装
3. 拖入原理图，命名为 J1

**第 2 步：放置排针**

1. 搜索 "HDR-M-2.54_1x8"（1x8pin 2.54mm 排针）
2. 拖入原理图，命名为 J2

**第 3 步：放置电容**

1. 搜索 "100nF 0603"
2. 拖入原理图，命名为 C1

**第 4 步：连线**

1. 根据第五节确认的 pin 映射表，用导线连接 J1 的对应 pin 到 J2 的对应 pin
2. 把 C1 连在 VCC 和 GND 之间
3. J1 上不用的 pin 标记为 NC（不连接）

**第 5 步：添加电源符号**

1. 添加 VCC 电源符号，连到所有 VCC 网络
2. 添加 GND 电源符号，连到所有 GND 网络

**第 6 步：检查**

1. 运行 ERC（电气规则检查）
2. 修复所有错误和警告

### 6.3 PCB 布局

**第 1 步：从原理图转到 PCB**

1. 点击 "设计" → "原理图转 PCB"
2. 所有元件会出现在 PCB 编辑器中

**第 2 步：设置板框**

1. 画一个矩形板框，建议尺寸：30mm x 18mm
2. 四个角加圆角（半径 1mm），美观且不扎手

**第 3 步：摆放元件**

```
板子布局（俯视图）：

┌──────────────────────────────────┐
│                                  │
│  ┌────────────────────────────┐  │
│  │  J1: FPC 50pin 连接器       │  │  ← 放在板子顶部
│  └────────────────────────────┘  │     排线从这里插入
│                                  │
│          [C1: 100nF]             │  ← 靠近 J1 的 VCC pin
│                                  │
│  ┌──┬──┬──┬──┬──┬──┬──┬──┐     │
│  │1 │2 │3 │4 │5 │6 │7 │8 │     │  ← J2: 排针放在板子底部
│  └──┴──┴──┴──┴──┴──┴──┴──┘     │     杜邦线从这里接出
│                                  │
└──────────────────────────────────┘
```

**第 4 步：布线**

1. 用铜线把 J1 的对应 pin 连到 J2 的对应 pin
2. 线宽设置：
   - VCC 和 GND：0.4mm（电源线要粗一点）
   - 信号线（DIN/SCLK/CS/DC/RST/BUSY）：0.25mm
3. 走线尽量短、直，避免交叉
4. 如果正面走不通，可以用过孔（via）走到背面

**第 5 步：铺铜**

1. 在 PCB 的背面（Bottom Layer）铺一整块 GND 铜皮
2. 这能减少电磁干扰，也方便 GND 连接

**第 6 步：检查**

1. 运行 DRC（设计规则检查）
2. 确保没有短路、没有未连接的网络
3. 目视检查走线是否正确

### 6.4 导出和打板

1. 点击 "制造" → "生成 Gerber"
2. 下载 Gerber 文件（一个 zip 包）
3. 打开嘉立创官网 https://www.jlc.com
4. 上传 Gerber 文件
5. 选择参数：
   - 层数：2 层
   - 板厚：1.6mm
   - 阻焊颜色：绿色（默认，最便宜）
   - 表面处理：HASL（有铅喷锡，最便宜）
6. 下单，5 片约 5 元，3-5 天到货

## 七、焊接指南

### 7.1 焊接顺序

从矮到高焊接，避免高元件挡住低元件：

```
第 1 步：焊 FPC 连接器 (J1)  ← 最矮，也是最难的
第 2 步：焊 100nF 电容 (C1)  ← 0603 封装，很小
第 3 步：焊 8pin 排针 (J2)   ← 最高，最简单
```

### 7.2 焊接 FPC 连接器（最难的部分）

FPC 连接器的 pin 间距只有 0.5mm，是整块板上唯一有难度的焊接。

**准备工作**：
1. 烙铁温度设到 320-350°C
2. 换上最细的烙铁头（尖头或刀头）
3. 在 PCB 焊盘上涂一层薄薄的助焊剂

**焊接步骤**：

```
步骤 1：固定连接器
  - 把连接器放到 PCB 焊盘上，对准位置
  - 先焊两端的固定脚（大焊盘），把连接器固定住
  - 确认连接器没有歪斜

步骤 2：焊信号 pin
  - 在所有 pin 上涂助焊剂
  - 用烙铁尖头轻触每个 pin，让焊锡流到 pin 和焊盘之间
  - 不需要每个 pin 都加焊锡，助焊剂 + PCB 焊盘上的预镀锡通常就够了
  - 如果焊锡太多导致相邻 pin 短路，用吸锡带清理

步骤 3：检查
  - 用放大镜（或手机微距模式）检查每个 pin
  - 确认没有虚焊（pin 和焊盘之间没有焊锡连接）
  - 确认没有短路（相邻 pin 之间有焊锡桥接）
  - 用万用表通断档逐个测量
```

**常见问题**：

| 问题 | 原因 | 解决方法 |
|------|------|---------|
| 相邻 pin 短路 | 焊锡太多 | 用吸锡带吸掉多余焊锡 |
| pin 虚焊 | 助焊剂不够或温度不够 | 补涂助焊剂，重新加热 |
| 连接器歪了 | 固定时没对准 | 用热风枪吹掉重来 |

### 7.3 焊接电容（简单）

0603 电容没有方向，正反都可以。

1. 在一个焊盘上先上一点焊锡
2. 用镊子夹住电容，放到焊盘上
3. 用烙铁加热有焊锡的那个焊盘，电容会被吸住
4. 焊另一端

### 7.4 焊接排针（最简单）

1. 把排针插入 PCB 的通孔
2. 翻到背面，逐个焊接
3. 每个焊点加热 2-3 秒，送入焊锡，形成圆锥形焊点

## 八、连接和测试

### 8.1 插入墨水屏排线

1. 掀开 FPC 连接器的翻盖（用指甲或镊子轻轻掀起）
2. 把墨水屏的 FPC 排线平插入连接器（注意触点朝向！）
3. 压下翻盖，锁紧排线
4. 轻轻拉一下排线，确认锁紧了不会脱落

**注意**：FPC 排线非常脆弱！
- 不要沿垂直方向弯折
- 不要反复弯折
- 不要向屏幕正面方向弯折

### 8.2 用杜邦线连接 ESP32

| 转接板排针 | 信号 | ESP32-S3 GPIO | 说明 |
|-----------|------|--------------|------|
| pin 1 | VCC | 3.3V | ESP32 开发板上标注 3V3 的引脚 |
| pin 2 | GND | GND | ESP32 开发板上任意 GND 引脚 |
| pin 3 | DIN | GPIO9 | 可以换成其他 GPIO |
| pin 4 | SCLK | GPIO10 | 可以换成其他 GPIO |
| pin 5 | CS | GPIO11 | 可以换成其他 GPIO |
| pin 6 | DC | GPIO12 | 可以换成其他 GPIO |
| pin 7 | RST | GPIO13 | 可以换成其他 GPIO |
| pin 8 | BUSY | GPIO14 | 可以换成其他 GPIO |

GPIO 编号可以自由选择，只要在固件代码中同步修改即可。
上面的编号和 InkTime 项目的 7.3 寸四色版固件一致，方便直接使用。

### 8.3 上电前检查

用万用表做以下检查，**不要跳过这一步**：

| 检查项 | 方法 | 正常结果 | 异常处理 |
|--------|------|---------|---------|
| VCC-GND 不短路 | 万用表电阻档测 VCC 和 GND | > 1kΩ | 如果接近 0Ω，说明有短路，检查焊接 |
| 信号线不短路 | 相邻信号 pin 之间测电阻 | 开路（无穷大） | 如果有阻值，说明 FPC 座焊接短路 |
| 排线连接正确 | 测 FPC 排线的 pin 到排针的通断 | 对应 pin 通断 | 如果不通，检查虚焊 |

### 8.4 烧录固件测试

1. 用 USB 线连接 ESP32 开发板到电脑
2. 打开 Arduino IDE
3. 安装 ESP32 Arduino Core 和 GxEPD2 库
4. 打开 InkTime 固件，修改屏幕型号和引脚定义
5. 编译烧录
6. 观察屏幕是否正常刷新显示

如果屏幕不亮，按以下顺序排查：

```
屏幕不亮排查流程：

1. ESP32 串口有输出吗？
   ├─ 没有 → ESP32 没正常启动，检查 USB 线和供电
   └─ 有 → 继续

2. 串口显示 "BUSY timeout" 吗？
   ├─ 是 → BUSY 线没接好，或 FPC 排线没插紧
   └─ 否 → 继续

3. 屏幕有任何反应吗（闪烁、变色）？
   ├─ 有但显示错误 → SPI 信号线可能接反了，检查 DIN/SCLK
   └─ 完全没反应 → 检查 VCC/GND 供电，检查 RST 线
```

## 九、常见问题

### Q1：我能不能不做 PCB，直接把线焊到 FPC 排线上？

不建议。FPC 排线的触点间距只有 0.5mm，直接焊线几乎不可能，而且会损坏排线。
FPC 连接器就是为了解决这个问题而存在的。

### Q2：我买的 FPC 连接器插不进排线怎么办？

检查三个参数是否匹配：pin 数（50）、间距（0.5mm）、上接/下接。
最常见的错误是买成了上接（排线触点朝上），但实际需要下接。

### Q3：能不能用微雪的通用驱动板（e-Paper Driver HAT）代替？

那块板的兼容列表里没有 7.3 寸 6 色 (E) 屏，不建议冒险。
而且它上面的 ESP32 是老款没有 PSRAM，内存可能不够。

### Q4：如果我不想自己焊 FPC 连接器怎么办？

可以找嘉立创的 SMT 贴片服务，让工厂帮你焊。
下单 PCB 时勾选"SMT 贴片"，上传 BOM 和坐标文件，工厂会帮你把 FPC 连接器焊好。
额外费用大约 20-30 元（含工费和元件费）。

### Q5：这块转接板能用于其他尺寸的微雪墨水屏吗？

不能直接用。不同尺寸的屏幕 FPC 排线的 pin 数不同（24pin / 50pin 等），
pin 映射关系也不同。但设计思路完全一样，改一下 FPC 连接器和走线就行。

## 十、嘉立创打板完整流程

### 10.1 从立创 EDA 导出 Gerber 文件

Gerber 文件是 PCB 工厂能看懂的"施工图纸"，包含每一层铜线、焊盘、阻焊、丝印的精确位置。

**导出步骤**：

1. 在立创 EDA 的 PCB 编辑器中，点击顶部菜单 "制造" → "生成制造文件(Gerber)"
2. 弹出的对话框中，保持默认设置即可
3. 点击 "生成 Gerber"，会下载一个 .zip 文件
4. 这个 zip 里包含多个文件，每个文件对应 PCB 的一层（见下方说明）

**Gerber 文件里有什么**：

| 文件后缀 | 对应的层 | 内容 |
|---------|---------|------|
| .GTL | 顶层铜 (Top Layer) | 正面的铜线走线和焊盘 |
| .GBL | 底层铜 (Bottom Layer) | 背面的铜线走线和焊盘 |
| .GTS | 顶层阻焊 (Top Solder Mask) | 正面的绿色油漆开窗（露出焊盘） |
| .GBS | 底层阻焊 (Bottom Solder Mask) | 背面的绿色油漆开窗 |
| .GTO | 顶层丝印 (Top Silkscreen) | 正面的白色文字标注（J1、C1 等） |
| .GBO | 底层丝印 (Bottom Silkscreen) | 背面的白色文字标注 |
| .GKO | 板框 (Board Outline) | PCB 的外形轮廓 |
| .DRL | 钻孔文件 (Drill) | 所有通孔和过孔的位置和直径 |

你不需要理解每个文件的内容，只要把整个 zip 上传到嘉立创就行。

### 10.2 嘉立创下单流程

**第 1 步：打开嘉立创官网**

访问 https://www.jlc.com ，注册账号（免费）。

**第 2 步：上传 Gerber 文件**

1. 点击首页的 "PCB 下单" 或 "立即下单"
2. 点击 "添加 Gerber 文件"，上传你导出的 zip 文件
3. 系统会自动解析并显示 PCB 预览图

**第 3 步：检查预览**

上传后会看到 PCB 的预览图，确认以下几点：
- 板子形状和大小是否正确
- 焊盘位置是否正确
- 丝印文字是否清晰

**第 4 步：选择参数**

| 参数 | 推荐选择 | 说明 |
|------|---------|------|
| 层数 | 2 层 | 我们的板子只需要 2 层 |
| 尺寸 | 自动识别 | 系统会从 Gerber 自动读取 |
| 数量 | 5 片 | 最小起订量，多的留着备用 |
| 板厚 | 1.6mm | 标准厚度，最便宜 |
| 阻焊颜色 | 绿色 | 默认，最便宜最快 |
| 丝印颜色 | 白色 | 默认 |
| 表面处理 | HASL（有铅喷锡） | 最便宜，焊接最方便 |
| 铜厚 | 1oz | 默认，够用 |

**第 5 步：确认价格和下单**

- 5 片 2 层板通常 ¥5-15（取决于板子大小）
- 选择快递方式
- 付款下单
- 通常 3-5 天到货

### 10.3 使用嘉立创 SMT 贴片服务

如果你不想自己焊 FPC 连接器（0.5mm 间距确实有难度），可以让嘉立创帮你焊。

**什么是 SMT 贴片**：

SMT = Surface Mount Technology（表面贴装技术）。就是工厂用机器把元件自动焊到 PCB 上。

**需要准备的文件**：

除了 Gerber 文件，SMT 贴片还需要两个额外文件：

| 文件 | 作用 | 如何生成 |
|------|------|---------|
| BOM（物料清单） | 告诉工厂每个位置放什么元件 | 立创 EDA → "制造" → "生成 BOM" |
| 坐标文件（Pick & Place） | 告诉机器每个元件放在 PCB 的哪个位置 | 立创 EDA → "制造" → "生成坐标文件" |

**BOM 文件内容示例**：

| 位号 | 数量 | 型号/值 | 封装 | 立创商城编号 |
|------|------|--------|------|------------|
| J1 | 1 | FPC 50P 0.5mm 下接 | — | C_XXXXX |
| C1 | 1 | 100nF | 0603 | C1591 |

**坐标文件内容示例**：

| 位号 | X (mm) | Y (mm) | 旋转角度 | 层 |
|------|--------|--------|---------|-----|
| J1 | 15.0 | 14.0 | 0 | Top |
| C1 | 12.0 | 8.0 | 90 | Top |

**下单步骤**：

1. 在嘉立创下单页面，PCB 参数选好后，勾选 "SMT 贴片"
2. 上传 BOM 文件和坐标文件
3. 系统会自动匹配元件（从立创商城库存中查找）
4. 确认每个元件都匹配成功
5. 选择贴片面（我们只需要贴正面 Top）
6. 确认价格（工费约 ¥20-30 + 元件费）
7. 下单

**注意**：排针（J2）是直插件，SMT 机器贴不了，需要你自己手焊。
但排针是最简单的焊接，完全没难度。

**费用估算**：

| 项目 | 费用 |
|------|------|
| PCB 打板（5 片） | ¥5-15 |
| SMT 工费 | ¥20-30 |
| FPC 连接器元件费 | ¥2-3 |
| 100nF 电容元件费 | ¥0.1 |
| 排针（自己焊） | ¥0.5 |
| 快递 | ¥10-15 |
| **总计** | **约 ¥40-65** |

这样你收到的板子上，FPC 连接器和电容已经焊好了，你只需要自己焊 8pin 排针就行。
