# 网页版相册需求文档

## 1. 项目概述

### 1.1 项目背景
InkTime 项目是一个基于 ESP32 和墨水屏的电子相册系统，目前主要功能包括照片分析、渲染和在墨水屏上显示。为了提供更丰富的用户体验，需要开发一个网页版相册，使用户可以通过浏览器浏览、搜索和管理照片。

### 1.2 项目目标
开发一个功能完善、美观的网页版相册，与现有的墨水屏电子相册形成互补，提供更加丰富的用户体验。

## 2. 功能需求

### 2.1 核心功能

| 功能模块 | 功能描述 | 优先级 |
|---------|---------|-------|
| 相册首页 | 照片网格展示，支持分页和排序 | 高 |
| 照片详情 | 单张照片详细信息，包括 EXIF 数据、评分等 | 高 |
| 分类浏览 | 按日期、类型、评分等分类浏览 | 中 |
| 搜索功能 | 搜索照片标题、描述等 | 中 |
| 响应式设计 | 适配电脑、平板、手机等设备 | 高 |

### 2.2 页面设计

| 页面名称 | 页面描述 | 优先级 |
|---------|---------|-------|
| 首页 | 照片卡片网格，支持瀑布流布局 | 高 |
| 详情页 | 大图展示，侧边信息栏，相关照片推荐 | 高 |
| 分类页 | 分类导航，筛选选项 | 中 |
| 搜索页 | 搜索框，搜索结果展示 | 中 |

## 4. 实现方案

### 4.1 服务器端实现

#### 4.1.1 路由模块

| 路由 | 方法 | 功能描述 |
|-----|------|---------|
| / | GET | 相册首页 |
| /photo/<path> | GET | 照片详情页 |
| /category | GET | 分类浏览页 |
| /search | GET | 搜索结果页 |
| /api/photos | GET | 获取照片列表 API |
| /api/photo/<path> | GET | 获取照片详情 API |
| /api/categories | GET | 获取分类列表 API |
| /api/search | GET | 搜索照片 API |

#### 4.1.2 模板系统

- **base.html**：基础模板，包含导航栏、页脚等公共部分
- **index.html**：相册首页，展示照片网格
- **photo.html**：照片详情页，展示单张照片和详细信息
- **category.html**：分类浏览页，按分类展示照片
- **search.html**：搜索结果页，展示搜索结果

### 4.2 前端实现

#### 4.2.1 页面设计

- **首页**：使用 Bootstrap 网格系统实现照片卡片网格，支持瀑布流布局
- **详情页**：左侧展示大图，右侧展示照片信息，包括 EXIF 数据、评分等
- **分类页**：顶部展示分类导航，下方展示分类照片
- **搜索页**：顶部展示搜索框，下方展示搜索结果

#### 4.2.2 交互功能

- **照片懒加载**：使用 Intersection Observer API 实现图片懒加载
- **无限滚动**：实现无限滚动加载更多照片
- **图片预览**：点击照片弹出预览模态框
- **平滑过渡**：添加平滑的页面过渡和交互动画

### 4.3 性能优化

- **图片优化**：使用适当尺寸的图片，支持 WebP 格式
- **缓存策略**：实现浏览器缓存和服务器缓存
- **分页加载**：使用分页或无限滚动加载照片
- **代码压缩**：压缩 CSS 和 JavaScript 文件

## 6. 部署计划

### 6.1 部署环境

- **操作系统**：Linux/Windows/MacOS
- **Python 版本**：3.8+
- **依赖项**：Flask、Bootstrap、SQLite

### 6.2 部署步骤

1. 安装依赖项：`pip install -r requirements.txt`
2. 配置环境变量：设置数据库路径、图片目录等
3. 启动服务器：`python src/server/app.py`
4. 访问网页：打开浏览器访问 `http://localhost:8765`

## 7. 风险评估

| 风险 | 影响 | 对策 |
|-----|------|------|
| 照片数量过多导致加载缓慢 | 影响用户体验 | 实现分页加载和图片懒加载 |
| 服务器资源不足 | 影响系统性能 | 优化代码，使用缓存机制 |
| 浏览器兼容性问题 | 影响部分用户 | 使用标准 Web 技术，测试主流浏览器 |
| 安全漏洞 | 影响系统安全 | 实现输入验证，防止路径遍历等攻击 |
