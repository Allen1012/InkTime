# 电路基础知识：TP4056 充电管理与电容原理

> 本文档是 InkTime PCB 设计学习过程中的基础知识整理，面向硬件零基础的开发者。

---

## 一、TP4056 锂电池充电管理芯片

### 1.1 锂电池为什么不能直接充电

锂电池充电必须遵循固定流程，不能简单地"通电就充"，否则会鼓包甚至爆炸。标准充电分三个阶段：

```
电池电压
4.2V ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ┐
                                    ╱         │ ③ 恒压阶段
                                  ╱           │ 电压恒定 4.2V
                                ╱             │ 电流逐渐减小
3.0V ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ╱               │ 直到电流 < 1/10
                            ╱                 │ 充电结束
2.9V ┐  ② 恒流阶段        ╱                  │
     │     电流恒定       ╱                   │
     │     电压缓慢上升  ╱                    │
     │                 ╱                     │
     │ ① 涓流 ────────╱                      │
     │  阶段                                  │
─────┴────╱──────────────────────────────────┴──→ 时间
     小电流   大电流（如 1A）    电流逐渐变小
```

| 阶段 | 条件 | 行为 | 目的 |
|------|------|------|------|
| ① 涓流 | 电池电压 < 2.9V | 用 1/10 电流慢充 | 保护过放的电池 |
| ② 恒流 | 2.9V → 4.2V | 恒定电流（如 1A）充电 | 主要充电阶段 |
| ③ 恒压 | 电压到达 4.2V | 保持 4.2V，电流自然减小 | 充满收尾 |

TP4056 就是自动完成这三个阶段的"充电管家"。

### 1.2 TP4056 芯片引脚

```
        ┌────────┐
  TEMP ─┤1      8├─ VCC (USB 5V 输入)
  PROG ─┤2      7├─ BAT (电池正极输出)
  GND  ─┤3      6├─ STDBY (充满指示)
  GND  ─┤4      5├─ CHRG (充电中指示)
        └────────┘
```

| 引脚 | 名称 | 功能 |
|------|------|------|
| 1 | TEMP | 温度检测（不用时接 GND 禁用） |
| 2 | PROG | 充电电流设定（接电阻到 GND） |
| 3, 4 | GND | 接地 |
| 5 | CHRG | 充电中指示（充电时拉低，驱动 LED） |
| 6 | STDBY | 充满指示（充满时拉低，驱动 LED） |
| 7 | BAT | 电池正极输出 |
| 8 | VCC | 电源输入（USB 5V） |

### 1.3 芯片内部原理

```
         TP4056 内部框图
┌─────────────────────────────────┐
│                                 │
│  USB 5V ──→ [电流调节器] ──→ BAT 输出
│              (恒流/恒压)        │
│                 ↑               │
│           [基准电压源]          │
│           (精确的 4.2V)         │
│                 ↑               │
│           [电压比较器]          │
│           (监测电池电压)        │
│                 ↑               │
│           [充电状态机]          │
│           (涓流→恒流→恒压→截止) │
│                 ↓               │
│           [状态输出]            │
│           (CHRG / STDBY)       │
└─────────────────────────────────┘
```

核心组件：
- **4.2V 基准电压**：锂电池的满充电压标准
- **电流调节器**：控制充电电流保持恒定
- **比较器**：监测电池电压，决定当前充电阶段
- **状态引脚**：输出充电状态（充电中 / 充满）

### 1.4 充电电流设定

只用一个电阻就能设定充电电流，接在 PROG 引脚和 GND 之间：

```
I_charge = 1200 / R_PROG

其中：I_charge 单位 mA，R_PROG 单位 Ω
```

| R_PROG | 充电电流 | 适用场景 |
|--------|---------|---------|
| 10kΩ | 120mA | 小电池（< 500mAh） |
| 2kΩ | 600mA | 中等电池 |
| 1.2kΩ | 1000mA (1A) | 18650 电池（InkTime 使用） |
| 1kΩ | 1200mA | 最大电流 |

### 1.5 InkTime 中的完整电路

```
USB 5V                                          电池 +
  │                                               │
  │    ┌────────────────────────────────┐          │
  ├───→┤ 8 (VCC)          7 (BAT) ├─────────┤
  │    │                                │          │
  │    │         TP4056                 │     ┌────┤
  │    │                                │     │    │
  │    ├─┤ 5 (CHRG) ──[1kΩ]──→ LED红   │  [10µF]  │
  │    │                                │     │    │
  │    ├─┤ 6 (STDBY) ─[1kΩ]──→ LED绿   │  ┌──┤    │
  │    │                                │  │  │    │
  │    ├─┤ 2 (PROG) ──[1.2kΩ]──→ GND   │  │  GND  │
  │    │                                │  │       │
  │    ├─┤ 1 (TEMP) ──────────→ GND    │  │       │
  │    │                                │  │       │
  │    ├─┤ 3 (GND)                      │  │       │
  │    ├─┤ 4 (GND) ───────────→ GND ───┘  │       │
  │    └────────────────────────────────┘   │       │
  │                                         │       │
  └──[4.7µF]──→ GND                        └───→ 到 LDO
```

| 元件 | 值 | 作用 |
|------|-----|------|
| R_PROG | 1.2kΩ | 设定充电电流为 1A |
| C_IN | 4.7µF | 输入滤波，稳定 USB 电源 |
| C_BAT | 10µF | 输出滤波，稳定电池端电压 |
| R_LED × 2 | 1kΩ | LED 限流电阻 |
| LED 红 | — | 充电中亮 |
| LED 绿 | — | 充满亮 |

### 1.6 充电保护

TP4056 本身只管充电，不管过放和短路。实际产品中通常串联 DW01A + 8205A 保护芯片：

```
TP4056 BAT 输出 → DW01A 保护板 → 电池
                    │
                    ├─ 过充保护（> 4.25V 断开）
                    ├─ 过放保护（< 2.5V 断开）
                    └─ 短路保护（电流过大断开）
```

InkTime 使用的 TP4056 成品模块（淘宝蓝色小板）已经集成了保护电路。

## 二、电容的作用

### 2.1 电容是什么

电容的本质是两块金属板中间夹一层绝缘体，能存储电荷。

```
      ┌──────────┐
  ──→ │ 金属板 A  │
      ├──────────┤  ← 绝缘层（电荷过不去，但电场能穿过）
  ←── │ 金属板 B  │
      └──────────┘
```

关键特性：**电容两端的电压不能突变**。就像水箱的水位只能慢慢升降，不能瞬间跳变。

### 2.2 水管类比

```
水泵（电源）──→ 水管 ──→ [弹性水囊] ──→ 水管 ──→ 水龙头（芯片）
                          (电容)

- 水泵压力波动时 → 水囊吸收波动，水龙头出水稳定
- 水龙头突然开大时 → 水囊释放存水，避免水压骤降
- 水泵关了 → 水囊里的存水还能维持一小会儿
```

### 2.3 在 InkTime PCB 上的三种用途

#### 用途 1：输入滤波（大电容 4.7µF / 10µF）

放在电源入口，平滑 USB 或电池的电压波动。

```
USB 5V ──→ [4.7µF] ──→ TP4056
              │
             GND
```

USB 电源不是完美直流，有纹波（微小电压波动）。大电容像大水箱，把波动"吃掉"。

没有它：TP4056 或 LDO 输入不稳定，可能工作异常。

#### 用途 2：去耦电容（小电容 100nF）

紧贴 ESP32 电源引脚放置，提供瞬时电流。

```
LDO 3.3V ──────────────────→ ESP32 VCC 引脚
                    │
                  [100nF]  ← 距离芯片引脚 < 3mm
                    │
                   GND
```

ESP32 在某些瞬间会突然需要很大电流（比如 WiFi 发射时从 20mA 跳到 300mA+），这个变化发生在纳秒级别。LDO 反应不过来（需要微秒级），电源线也有电感阻碍电流突变。

100nF 电容就是芯片旁边的"应急电池"：

```
0ns     ESP32 突然需要 300mA
        ↓
0-10ns  100nF 电容释放电荷，维持电压
        ↓
10ns+   LDO 稳压器反应过来，开始供电
        ↓
        电容重新充满，等待下一次
```

没有它：ESP32 电源引脚电压瞬间跌落（3.3V → 2.8V），可能导致 WiFi 断连、程序跑飞、芯片复位。

#### 用途 3：稳压器输出电容（1µF）

放在 LDO 输出端，配合稳压器内部反馈环路工作。

```
LDO MIC5219
  IN ←── 3.7V
  OUT ──→ 3.3V ──→ [1µF] ──→ ESP32
  GND              │
                  GND
```

这是 MIC5219 数据手册要求必须加的，容量按手册推荐值选择。

### 2.4 为什么不同位置用不同大小

| 位置 | 容量 | 原因 |
|------|------|------|
| 电源入口 | 4.7µF - 10µF | 大水箱，平滑低频波动 |
| 芯片旁边 | 100nF | 快速响应高频瞬变，小电容响应更快 |
| LDO 输出 | 1µF | 数据手册指定，配合内部反馈环路 |

规律：**电容越小响应高频越好，电容越大储能越多但响应慢**。实际设计中大小电容并联——大的管低频，小的管高频。

```
InkTime 电源滤波链路：

电池 → [10µF] → LDO → [1µF] → [100nF] → ESP32
       大水箱    稳压   配合LDO   应急电池
```

### 2.5 电容的物理参数

InkTime BOM 里的电容都是 0603 封装（1.6mm × 0.8mm），比芝麻还小。

选型关注两个参数：
- **容量**：µF / nF（能存多少电荷）
- **耐压**：能承受的最大电压（InkTime 用 3.3V/5V 系统，选 10V 或 16V 耐压即可）

### 2.6 一句话总结

电容在 PCB 上的核心作用是**让电压保持稳定**——大电容平滑电源波动，小电容应对芯片瞬时用电需求。没有电容的电路就像没有减震器的车，路面一颠簸（电流一变化）整个车（芯片）都在抖。
